# DermExpert: Rule-Based Skin Disease Diagnosis Expert System

**Group Name:** BRAINIES

## Project Overview

DermExpert is a **rule-based expert system** designed to assist in the diagnosis of common skin diseases. The system simulates the decision-making process of dermatologists using an **inference engine** based on IF–THEN rules. Users input symptoms, and the system evaluates them to generate likely diagnoses, severity levels, and explanations.

This system is developed for academic purposes and demonstrates how expert systems can be applied in healthcare to provide consistent and transparent preliminary diagnoses.

## Key Features

- Forward chaining inference engine  
- Transparent reasoning with explanation module  
- Diagnoses common skin conditions such as:
  - Acne  
  - Eczema  
  - Psoriasis  
  - Ringworm (Fungal Infection)  
  - Dermatitis  
  - Scabies  
- Provides severity level (Mild, Moderate, Severe)  
- Preliminary advice based on diagnosis  

## System Scope

The system evaluates patient-reported symptoms including:

- Itching  
- Redness  
- Rashes  
- Swelling  
- Skin peeling  
- Pus  
- Duration of symptoms  
- Location of affected area  

## Technologies Used

- Frontend: React or HTML/CSS/JavaScript
